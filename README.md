# doctor-booking-portal
Led end-to-end UI/UX design for Unicare as the sole designer, creating a seamless booking experience across desktop and mobile. Built and maintained the design system and tokens, while designing portals for patients, doctors, and clinic admins in close collaboration with developers.

Unicare — Doctor Booking Platform
Overview
Unicare is a multi-platform doctor booking system designed to simplify how patients schedule appointments while enabling doctors and clinic administrators to efficiently manage their operations. The platform addresses the need for a clear, seamless, and scalable booking experience across desktop and mobile.
My Role
UI/UX Designer — responsible for end-to-end design including user research, wireframing, prototyping, and final UI.
As the sole designer, I also built and maintained the design system and design tokens, and collaborated closely with developers to ensure consistent implementation across the platform.
The Problem
Booking medical appointments can often feel fragmented and confusing, with unclear steps, inconsistent interfaces, and inefficiencies across different user roles. Patients struggle with navigation and decision-making, while doctors and clinic administrators lack streamlined tools to manage schedules and operations within a unified system.
Process
1. User Research
Conducted user testing to identify friction points in the booking flow, focusing on where users experienced confusion or drop-off. Insights gathered helped define key usability issues and opportunities for improvement.
2. Wireframing
Developed low-fidelity wireframes to map out clear and intuitive user flows for booking, scheduling, and management features across all user types.
3. Prototyping
Created interactive prototypes in Figma to simulate real user interactions, allowing for early validation of flows and design decisions.
4. Testing & Iteration
Iteratively refined the design based on user feedback, improving clarity, reducing cognitive load, and ensuring a more predictable and seamless experience across all touchpoints.
